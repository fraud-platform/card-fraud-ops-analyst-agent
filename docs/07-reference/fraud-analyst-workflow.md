# Fraud Analyst Workflow Reference

## Analyst Goals

- Prioritize investigation queue efficiently.
- Validate evidence quality, not just model output.
- Document decisions and escalations.
- Propose or request rule changes through governance.

## Where Ops Agent Helps

1. Summarizes cross-signal evidence quickly.
2. Highlights likely high-impact investigations.
3. Suggests actionable next steps.
4. Generates rule draft packages for analyst review.

## Human Control Points

- Analyst decides whether recommendation is valid.
- Analyst decides whether to escalate.
- Analyst decides whether to export rule draft.
- Rule activation remains in Rule Management maker-checker process.

## Quality Metrics Aligned to Workflow

- Reduced median investigation time.
- Improved true-positive ratio on analyst-handled alerts.
- Stable or improved false-positive handling quality.
